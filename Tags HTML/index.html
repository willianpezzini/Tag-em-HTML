O que são tags?

(Voltar)

Os arquivos com a extenção .HTML, são os responsáveis por estruturar a estrutura de uma página Web. As tags HTML, também podem ser chamadas de elementos, e tem como principal função, fazer informar para o navegador como ele deve interpretar a página, mostrando a ordem de exibição e identificando configurações padrão de cada tag.

Tipos de Tags

(Voltar)

Inicialmente pode-se definir dois tipos de Tags:

Block-level => Cria um bloco ou container que prenche toda a largura do elemento.

Inline => Usado para demarcação de conteúdos de texto, muitas vezes usado para criar links para acesso rápido na propria página.

Principais Tags

(Voltar)

  1. html => É a raiz do Documento. Envolve todos os outros elementos.

  2. head => Recebe as informações que serão interpretadas pelo navegador. Dentro dela se encontram tags inportantes como o title e a tag meta.

  3. body => É o corpo do documento. Indica que tudo que está ali dentro desta tags será exibido na tela para o usuário.

Tags Semanticas

(Voltar)

São tags que dão sentido para as informações inseridas nelas.

Estas são as principais tags semanticas:

  • header = Inclui os elemento do cabeçalho.
  • main = Define o conteúdo dentro dele como sendo o conteúdo principal.
  • footer = Insere uma nota de roda pé.
  • nav = Cria uma barra de navegação.
  • ol = Cria uma lista ordenada.
  • ul = Cria uma lista NÃO ordenada.
  • li = Cria os itens de ambas as listas
  • p = Cria um paragrafo no texto.

Tags semanticas textuais

  • mark = Serve para destacar uma parte do texto, ficando marcado como caneta marca texto.
  • small = Formata o texto para letras peguenas, muito utilizado para notas de roda pé.
  • i = Deixa o texto em Itálico.
  • u = Deixa o texto sublinhado.
  • strong = Deixa o Texto em Negrito.
  • a = Cria um texto ancora, permite que o usuário clique sobre o texto possa ser redirecionado para outra parte ou página.
  • hr = Cria uma linha horizontal por toda a extenção da tela.
  • sub = Cria textos Subescritos.
  • sup = Cria textos Sobrescritos.
  • blockquote = Cria uma citação.
